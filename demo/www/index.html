A browser engine, running inside your browser.

Everything below this line is rendered by Manuk — a from-scratch engine in Rust — compiled to WebAssembly and executing on this page, right now. Scroll it. Hover it. It is laying the document out as you do.

Real: Stylo (the actual cascade), Taffy (the actual flex/grid), tiny-skia (the actual rasterizer), and this engine's own DOM, layout and paint. Scrolling re-renders — it does not pan a bitmap.
Not real, and worth saying: no JavaScript (SpiderMonkey is C++ and does not target wasm), and no live URL fetching — these are bundled snapshots of real pages. Saying so is the only thing that makes the rest of it believable.
